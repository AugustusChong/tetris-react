# Tetris React
This is a Tetris clone with a scoreboard and keyboard controls. 

## Tech/Framework Used
Built with React, Redux Toolkit, SASS.

## Controls
<pre>
Left Arrow:  Move Left
Right Arrow: Move Right
Down Arrow:  Move Down
Up Arrow:    Rotate
C:           Hold/Switch Tetromino
</pre>
